# evercam-playbook

Coming soon to the theaters near you...

Goals
 * Setup aws servers in a single command for
    * evercam-media
    * evercam-api
    * evercam-dash
 * Deploy to different environments
    * live
    * stage
    * demo
    * ... whatever you name it.
 * Optionally setup (for working on local)
    * postgres database
    * redis
    * memcached
